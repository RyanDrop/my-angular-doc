Property binding

O property binding (vinculação de propriedade) em Angular ajuda a definir valores para propriedades de elementos HTML ou diretivas. Use a associação de propriedade para funcionalidade do botão de alternância, definir caminhos programaticamente, compartilhar valores entre componentes e etc.

Three-Four

Muitas vezes , a interpolação e a vinculação de propriedades podem obter os mesmos resultados. Os pares de ligação a seguir fazem a mesma coisa. É o caso das imagens, no entanto, ao definir uma propriedade de elemento para um valor de dados não string, deve-se usar a associação de propriedade.
